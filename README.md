# Agentic SWE Training

> I will update this project base on my actual hands-on experience.

Teaching and demonstration repository for learning how to work effectively with agentic AI systems like Claude Code.

## Purpose

This repository serves as a sandbox and reference for peer software engineers learning to leverage agentic AI in their daily work. It contains guides, examples, and demo projects that showcase practical AI-assisted workflows.

## Structure

```
.
├── guides/           # How-to guides and workflows
├── examples/         # Utility scripts and code examples
├── calculator/       # Demo Laravel application
└── README.md         # This file
```

### Directories

- **`guides/`** - Step-by-step guides for specific workflows and concepts
- **`examples/`** - Reusable scripts, patterns, and code snippets
- **`expense-tracker/`** - Demo project for hands-on practice

## Getting Started

1. Clone or navigate to this repository
2. Explore the `guides/` directory for workflow documentation
3. Check `examples/` for utility scripts
4. Use demo projects like `expense-tracker/` for hands-on learning
